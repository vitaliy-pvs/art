Отображение Unicode emoji в поле ввода "Глубина шкафа" при вводе пользователем

Version - версия ПО
Environment - ОС + версия, модель устройства
Severity - серьезность
Priority - приоритет

Предусловие.
А. Страница https://mchk.pythonanywhere.com/ перезагружена
Б. Работа с конфигуратором МШК-01

Шаги воспроизведения.
1. Для параметра "Глубина шкафа" выбрать значение unicode emoji "grinning face"

Фактический результат.
А. Unicode emoji "grinning face" отображается в поле значения параметра "Глубина шкафа"
Б. В области уведомлений отображается текст "Замечаний нет."
В. В таблице раздельной выписки в поле "Глубина шкафа" отображается значение Unicode emoji "grinning face"

Ожидаемый результат.
А. Unicode emoji "grinning face" отображается в поле значения параметра "Глубина шкафа"
Б. Поле значения параметра "Глубина шкафа" выделено красным цветом
В. В области уведомлений отображается текст "- Укажите глубину шкафа не менее глубины механизма и не более 600 мм."
Г. Таблица раздельной выписки и кнопка "Скачать Excel" скрыты.
